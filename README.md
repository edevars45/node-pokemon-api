# API REST Pokémon avec Node.js et Express

Projet de formation pour apprendre à créer une API REST avec Node.js et Express.js.

## Technologies utilisées

- **Node.js** : Environnement d'exécution JavaScript côté serveur
- **Express.js** : Framework web pour créer des APIs REST
- **Nodemon** : Outil de développement pour redémarrer automatiquement le serveur

## Installation

### Prérequis
- Node.js installé (version 18 ou supérieure)
- NPM installé
